JavaScript Statements

A JavaScript program is a list of statements to be executed by a computer.

In a programming language, these programming instructions are called statments.

JavaScript program is a list of programming statements. Most JavaScript programs contain many JavaScript statments. The statments are executed, one by one, in the same order as they are writte.

JavaScript programs (and JavaScript statements) are often called JavaScript code.

JavaScript White Space

JavaScript ignores multiple spaces. You can add white space to your script to make it more readable.

var person = "Hedge"; is easier to read than var person="Hedge";

A good practice is to put spaces around operators ( = + - * / ):

var x = y + z;

JavaScript Line Length and Line Breaks

For best readability, programmers often like to avoid code lines longer than 80 characters.

If a JavaScript statment does not fit on one line, the best place to break is after an operator.

JavaScript Keywords

JavaScript statements often start with a keyword to identify the JavaScript action to be performed.

JavaScript keywords are reserved words. Reserved words cannot be used as names for variables.

Below is a table with some keywords:

Keyword Description
break Terminates a switch or aloop.
continue Jumps out of a loop and starts at the top.
debugger Stops the execution of JavaScript, and calls (if available) the debuggin function.
do...while Executes a block of statements to be executed, as long as a condition is true.
for Marks a block of statements to be executed, depending on a condition.
function Declares a function.
if...else Marks a block of statements to be executed, dpending on a condition.
return Exits a function.
switch Marks a block of statments to be executed, depending on a different cases.
try...catch Implements error handling to a block of statements.
var Declares a variable.
